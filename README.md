# conkify
Conky setup for showing Spotify album artwork on the desktop

## Dependencies

* bash
* wget
* dbus
* conky
* sp (included)

## Installation

Simply copy the contents of the repo to $HOME/.conky, ensure Spotify is running, and invoke conky with:
```
$ conky -c $HOME/.conky/conkyrc
```
